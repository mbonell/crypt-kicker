Crypt Kicker
============
MeÌtodo popular pero inseguro para encriptar texto que consiste en permutar las letras del alfabeto. Esto es, en el texto, cada letra del alfabeto es consistentemente reemplazada por alguna otra letra. Para asegurar que el encriptado es reversible, no se permite que dos letras sean reemplazadas por la misma letra.
